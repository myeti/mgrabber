# MGrabber

## Installation

```
npm install
```

## Fetch manga list

```
node grab.js -fetch
```

## Download a specific manga

```
node grab.js beelzebub
```

## Download a specific manga AND build it into pdf

```
node grab.js beelzebub -pdf
```

To chunk into multiple pdf file:

```
node grab.js beelzebub -pdf=10
```
